Project: BB-8

Wie ben ik?

Ik ben Sander Peters, ben eerstejaars student op de studie Kunst en Techniek. Mijn passie ligt in het technische gedeelte, de software en iets fysieks maken. Mijn talent ligt bij mijn passie, en door dit project is die alleen nog maar groter geworden.

Het Project

In dit project maak ik BB-8, de iconsiche droid uit de nieuwste StarWars film. Ik heb voor deze droid gekozen omdat ik graag techische en software wou combineren, dit samen met een fysiek object maken lijd snel tot robot. Dit heb ik gecombineerd met mijn grote liefde voor StarWars en snel kwam ik tot het idee voor BB-8

BB-8

BB-8 beweegt op jouw commando, met een controller kan je hem simpel besturen, je kan voor en achteruit, hij kan 360 graden rond draaien zonder draaicirkel en het beste van alles, hij maakt de orginele geluiden uit de film. Je was waarschijnlijk al verliefd toen je BB-8 in de film zag, wacht maar tot je hem echt ziet.
